# Matissa and Exponent

$a \times 10^{-b}$
- $a$ - matissa
- $10^{-b}$ - exponent


# Single vs Double



## Single 

32 bits
from left to right
1st bit           - sign
next 8 bits   - exponent
next 23 bits - matissa

## Double 
64 bits
from left to right
1st bit           - sign
next 11 bits   - exponent
next 52 bits - matissa
